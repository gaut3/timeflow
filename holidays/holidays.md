# Fremtidige dager

Dette dokumentet inneholder en oversikt over fremtidige dager som skal påvirke dashboardet. Dette inkluderer både offentlige fridager (helligdager) og planlagte fridager.
## Hvordan legge til egne planlagte dager

Du kan legge til dine egne planlagte dager ved å følge samme format som helligdagene:

```
- YYYY-MM-DD: type: Beskrivelse
```

**Tilgjengelige typer:**
- `helligdag`: Offentlige fridager (røde dager)
- `ferie`: Planlagt ferie
- `avspasering`: Planlagt avspasering
- `velferdspermisjon`: Planlagt velferdspermisjon
- `studie`: Planlagt studiedag
- `kurs`: Planlagt kursdag
- `half`: Halv arbeidsdag (reduserer ukemålet med halv dag, 3.75 timer)

**Halve arbeidsdager:** Du kan bruke `half:` som type for halve arbeidsdager, eller legge til `:half` etter en annen type (f.eks. `ferie:half:`).

**Eksempel:**

```
- YYYY-MM-DD: ferie: Sommerferie uke 1
- YYYY-MM-DD: ferie: Sommerferie uke 1
- YYYY-MM-DD: avspasering: Ekstra fridag
- YYYY-MM-DD: kurs: AWS sertifisering
- YYYY-MM-DD: half: Romjul (halv arbeidsdag)
- YYYY-MM-DD: half: Julaften (halv arbeidsdag)
- YYYY-MM-DD: ferie:half: Halv feriedag (alternativ syntaks)
```
